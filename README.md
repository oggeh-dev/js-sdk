# OGGEH JavaScript SDK

## Build

```bash
yarn build
```

## Usage

### Browser

```html
<script src="https://unpkg.com/@oggeh-dev/js-sdk/build/oggeh.browser.min.js"></script>
```

### ES Module

```javascript
import { OGGEH } from "@oggeh-dev/js-sdk";
```

### CommonJS

```javascript
const { OGGEH } = require("@oggeh-dev/js-sdk");
```
